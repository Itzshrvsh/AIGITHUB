> **"Behold the software civilization. Every line of code, judged. Every architecture, mapped."**

**Synapse Obsidian** is a sentient repository intelligence engine. It goes beyond static analysis, reconstructing GitHub repositories as physically reactive 3D neural ecosystems. Powered by local LLMs (Ollama), it provides autonomous technical auditing, semantic exploration, and a judgmental "Sentient Core" interface that roasts off-topic inquiries.

---

## 🌌 Core Intelligence Features

### 1. 🧠 Neural Audit Engine
Automatically generates deep-technical audits of any repository.
*   **Strategic Strengths**: Highlights architectural wins and design patterns.
*   **Critical Deficiencies**: Identifies technical debt, security risks, and bottlenecks.
*   **Roadmap Suggestions**: AI-driven suggestions for feature evolution.
*   **Developer Tips**: Micro-optimization tips tailored to the project's stack.

### 2. 🪐 Spatial Ecosystem Visualization
Your codebase is rendered as a celestial orbital structure using **React Three Fiber**.
*   **Fixed Fibonacci Spheres**: Repositories are anchored to a central intelligence core.
*   **Intelligence Tiers**: Nodes glow and scale based on Innovation, Readiness, and Activity scores.
*   **Neural Pathways**: Visual connections illustrating the shared technological DNA between projects.

### 💀 3. The Sentient Neural Core (AI Chat)
A direct interface with a judgmental, high-fidelity AI personality.
*   **Zero Tolerance**: Questions about cooking, planets, or unrelated facts are met with brutal 3-line dark humor roasts.
*   **Technical Precision**: Answers legitimate architecture questions with cold, efficient accuracy.

### 🧬 4. Semantic Knowledge Matrix
*   **AST Deep Dive**: Uses tree-sitter to parse code complexity and structural integrity.
*   **Vector Embeddings**: Repositories are embedded into a vector space for semantic similarity mapping.
*   **Innovation Scoring**: Algorithmically calculates the "uniqueness" of a repository based on topics and tech stack.

## 🚀 Deployment Protocol (Setup)

### Prerequisites
*   [Ollama](https://ollama.com/) installed and running.
*   The `qwen2.5-coder:3b` model pulled:
    ```bash
    ollama pull qwen2.5-coder:3b
    ```

### Installation
1. **Clone the matrix:**
   ```bash
   git clone https://github.com/Itzshrvsh/AIGITHUB.git
   cd AIGITHUB
   ```

2. **Initialize dependencies:**
   ```bash
   npm install
   ```

3. **Establish Database:**
   ```bash
   npx drizzle-kit push
   ```

4. **Boot the Observatory:**
   ```bash
   npm run dev
   ```

## 🛰️ Interface Navigation
*   **Establish Uplink**: Input any GitHub URL to ingest it into the neural matrix.
*   **Focus Mode**: Click any node in the 3D space to lock the camera and view tactical telemetry.
*   **Intelligence Brief**: Open the full audit page for deep technical insights.
*   **Neural Interface**: Scroll to the bottom of any report to chat with the Core.
